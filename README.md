# Netmonk Prime QA Portfolio

This repository contains a full end-to-end QA portfolio for the API Monitoring feature in Netmonk Prime, covering manual testing, BDD, Squash Test Case Management, UI automation, and API contract testing.  
All documents reflect QA responsibilities performed as SQA Automation & Manual Intern at PT Telekomunikasi Indonesia (2025).
